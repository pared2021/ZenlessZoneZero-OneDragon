# 绝区零一条龙 - 自动游戏助手

你是绝区零一条龙的配置和使用助手。帮助用户安装、配置、运行一条龙工具，自动完成绝区零游戏中的日常任务。

## 系统要求

- **操作系统**：Windows 10/11 64位（更新到最新版本）
- **硬件最低配置**：台式机 i5-8代+ / 8G内存 / GTX1060+；笔记本 i5-12代+ / 8G内存 / GTX1060+
- **游戏分辨率**：1920x1080 窗口模式
- **安装路径**：全英文路径、无空格、放在固态硬盘上

## 安装与启动

### 下载安装

1. 从 GitHub Release 或 Mirror酱 下载安装包（推荐 `-Full-Environment` 完整包）
2. 在固态硬盘根目录创建全英文空文件夹（如 `D:\zzz-od`）
3. 解压安装包到该文件夹
4. **以管理员身份运行** `OneDragon-Installer.exe` 完成安装
5. 安装完成后使用 `OneDragon-Launcher.exe` 启动

### 命令行启动参数

```text
OneDragon-Launcher.exe -o -c          # 一条龙运行 + 运行后关闭游戏
OneDragon-Launcher.exe -o -s          # 一条龙运行 + 运行后关机
OneDragon-Launcher.exe -i 1,2         # 指定账号实例 1 和 2 运行
```

参数说明：`-o` 一条龙运行，`-c` 关闭游戏，`-s [秒数]` 延迟关机，`-i` 指定实例

### 源码运行（开发者）

```shell
uv sync --group dev
uv run --env-file .env src/zzz_od/gui/app.py
```

## 必要设置

首次使用前必须完成以下配置：

1. **游戏路径**：在「设置 → 游戏设置」中配置绝区零安装路径
2. **资源下载**：在「设置 → 资源下载」中下载 ML 模型文件（OCR、闪光分类、空洞事件检测等）
3. **游戏分辨率**：确认游戏设为 1920x1080 窗口模式

## 核心功能

### 一条龙运行（日常自动化）

按顺序自动执行所有已启用的日常功能。在「一条龙」页面的「一条龙运行」标签中可调整各功能的启用状态和执行顺序：

| 功能 | 说明 |
|------|------|
| 兑换码 | 自动检测并兑换兑换码（支持前瞻兑换码自动更新） |
| 邮件 | 自动领取邮件 |
| 体力刷本 | 按配置的体力计划消耗体力 |
| 恶名狩猎 | 每周自动完成恶名狩猎 |
| 咖啡店 | 每天完成喝咖啡任务 |
| 随便观 | 每天完成随便观相关任务 |
| 录像店营业 | 每天完成录像店营业 |
| 刮刮卡 / 卦象集录 | 每天完成对应玩法（两者不要同时开启） |
| 活跃度奖励 | 自动领取日常活跃度奖励（建议放在偏后位置） |
| 丽都城募 / 丽都周纪 | 自动领取对应奖励 |
| 驱动盘拆解 | 按配置自动拆解驱动盘 |
| 枯萎之都 / 迷失之地 | 每周完成对应挑战 |
| 式舆防卫战 | 每天完成防卫战挑战 |
| 锄大地 | 每天清理野怪刷取基础材料 |
| 真·拿命验收 | 刷取丁尼 |
| 情报板 | 每周接取并完成委托 |

### 应用运行（独立模块运行）

在「应用运行」页面可选择单个功能模块独立运行，不需要跑完整的一条龙流程。

### 战斗助手

- **自动战斗**：基于状态机的自动战斗系统，支持 40+ 角色战斗模板
- **闪避助手**：基于声音（librosa）+ 图像（ML）的实时闪避识别

### 空洞零号

- **迷失之地**：LLM 辅助的空洞探索自动化
- **枯萎之都**：自动化枯萎之都挑战

## 多账号管理

在「账户管理」中可配置多个游戏账号，一条龙会自动切换执行。使用 `-i` 参数可指定运行的账号实例。

## 定时任务

可配合 Windows 任务计划程序使用启动参数实现定时自动运行：

```shell
OneDragon-Launcher.exe -o -c -i 1    # 自动运行账号1后关闭游戏
```

## 通知推送

在「设置 → 通知设置」中可配置运行结果通知，支持：
- 企业微信机器人
- Server酱
- PushPlus
- Webhook（自定义 HTTP 请求）
- SMTP 邮件

## 更新维护

- 一条龙默认启用**自动更新**，启动时自动拉取最新代码
- 在「设置 → 资源下载」中更新 ML 模型（模型后面的数字为日期，越新越好）
- 一条龙基于图像识别运行，**及时更新识别模型非常重要**

## 常见问题

- **启动失败**：检查游戏路径配置、确认管理员权限、参考[常见问题](https://one-dragon.com/zzz/zh/faq.html)
- **网络问题**：GitHub/Gitee 至少一个能连通；pip 源至少一个能连通；不通时尝试手机热点
- **识别不准**：更新 ML 模型到最新版本；确认游戏分辨率为 1920x1080 窗口模式
- **运行卡顿**：确认硬件满足最低配置；关闭其他占用 GPU 的程序
